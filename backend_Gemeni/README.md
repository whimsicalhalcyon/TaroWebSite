## Получение API‑ключа Gemini

1. Перейдите по ссылке для создания API‑ключа:  
   https://aistudio.google.com/app/apikey?hl=ru
   
***!!! Gemeni не доступен в РФ используй VPN\VLESS !!!***

В бесплатном тарифе доступно достаточное количество запросов в минуту и токенов для работы.

2. Создайте файл `.env` в корневой папке проекта и добавьте следующие переменные:

   ```env
   GEMINI_API_KEY="ваш_ключ"
   GEMENI_MODEL="gemini-2.5-flash-lite"
   ```

## Сборка и запуск контейнера

1. Откройте терминал в корневой папке проекта.

2. Соберите Docker‑образ (с опцией `--no-cache` для чистой сборки):

   ```
   docker-compose build --no-cache
   ```

3. Запустите контейнеры в фоновом режиме:

   ```
   docker-compose up -d
   ```

## Управление контейнерами

- **Остановка и удаление контейнеров:**

  ```
  docker-compose down
  ```

- **Просмотр логов:**

  ```
  docker-compose logs
  ```

  Также логи можно посмотреть в интерфейсе **Docker Desktop**.

## Примечания

- Убедитесь, что Docker установлены и корректно работают на вашей системе.
- Сохраняйте файл `.env` в секрете — не выкладывайте его в публичные репозитории.
- Для изменения модели или API‑ключа редактируйте файл `.env`.
